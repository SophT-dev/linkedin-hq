# make-webhook

This hook fires a Make (formerly Integromat) webhook when Claude Code stops, posting the session context as the trigger payload. It connects Claude Code to any Make scenario that starts with a Custom Webhook module, enabling agent outputs to flow into Make's visual automation builder.

## When it fires

Fires on the `Stop` event by default. You can change the event in `hook.json` to `SessionEnd`, `PostToolUse`, or any other valid Claude Code hook event depending on when your Make scenario should be triggered.

## What it runs

```
orchestrator/run.sh make fire
```

The script reads the session context from stdin and sends an HTTP POST to the configured Make webhook URL.

## Required environment variables

| Variable | Description |
|---|---|
| `MAKE_WEBHOOK_URL` | The webhook URL generated by the Custom Webhook module in your Make scenario |

## Optional environment variables

| Variable | Description |
|---|---|
| `MAKE_PAYLOAD_FILTER` | Comma-separated list of session context keys to include in the payload (defaults to sending everything) |
| `MAKE_WEBHOOK_SECRET` | Shared secret sent as `X-Make-Secret` header for payload validation in the scenario |
