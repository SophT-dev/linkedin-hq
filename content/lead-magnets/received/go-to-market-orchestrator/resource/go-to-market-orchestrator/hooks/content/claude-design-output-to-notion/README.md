# claude-design-output-to-notion

This hook saves design outputs produced during a session to a Notion database when Claude Code stops. It captures design briefs, component specifications, layout descriptions, and other design artefacts generated by the agent, keeping them accessible alongside other project documentation in Notion.

## When it fires

Fires on the `Stop` event, which Claude Code triggers whenever the agent stops generating output and returns control to the user.

## What it runs

```
orchestrator/run.sh notion save-design
```

The script identifies design-related content in the session output and creates Notion pages in the configured design archive database.

## Required environment variables

| Variable | Description |
|---|---|
| `NOTION_API_KEY` | Internal integration token from your Notion integration settings |
| `NOTION_PARENT_PAGE_ID` | Page ID under which new design pages are created |
