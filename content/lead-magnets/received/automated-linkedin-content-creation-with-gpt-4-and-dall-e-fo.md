# Automated LinkedIn Content Creation with GPT-4 and DALL-E for Scheduled Posts

**Source note:** n8n community workflow template.


## Key takeaways
- n8n workflow chain: Schedule Trigger -> GPT-4 generates trending topics + writes the post in a custom voice -> DALL-E generates a matching image -> Merge node -> LinkedIn API posts automatically.
- Creator-reported results: '400% increase in profile views within 3 weeks', scaling from ~12 manual posts/month to 120+ posts/month.
- Setup requirements: OpenAI API key, LinkedIn API access, customizable prompts for voice/industry matching, optional human-approval step before posting.
- Estimated setup time 30-45 minutes; estimated running cost $5-15/month in OpenAI API fees.

## Excerpt
Automatically generates trending LinkedIn content topics using AI... Writes posts in your authentic voice using OpenAI... Creates professional images with DALL-E... Posts everything on schedule without manual intervention.
