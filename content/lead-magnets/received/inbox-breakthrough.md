# Inbox Breakthrough

**Source note:** RevGen Labs.


## Key takeaways
- Core claimed results: 90%+ primary inbox placement, 15-25% reply rates, 50+ meetings/month per client, scaling to thousands of daily emails without getting flagged.
- Domain rule: .com only ('everwarm' if 2+ years old); buy in batches of 3-4 spread over weeks, let sit 3+ weeks before use, max 2 inboxes per domain.
- Sending caps: 20-25 emails/day per inbox including follow-ups; scale via more inboxes, not more volume per inbox (example: 1000 inboxes set up for one client).
- Warm-up ratio rule: 40% regular conversations / 40% outreach campaigns / 20% warm-up-maintenance to protect sender reputation.
- Explicit week-by-week scaling schedule: Weeks 1-4 warm-up only, Week 5 at 10-15/day, Weeks 6-8 +5/week, Week 9+ capped at 25-30/day per inbox.
- Crisis thresholds: bounce rate >5% or >1 spam complaint per 1,000 emails means stop sending immediately, run blacklist checks, rest problem inboxes 2-3 weeks.
- Matching rule: match sending provider to recipient's provider (Gmail-to-Gmail, Outlook-to-Outlook), claimed +15-20% deliverability overnight; recommends plain-text-only emails as the 2025 standard.
